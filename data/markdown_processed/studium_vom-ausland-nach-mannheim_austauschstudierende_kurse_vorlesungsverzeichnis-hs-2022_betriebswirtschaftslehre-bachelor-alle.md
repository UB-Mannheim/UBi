# Austauschkurse Business School Mannheim

## Für Studierende im Austausch

Die Wahl der geeigneten Kurse hängt von Ihrem aktuellen Studienniveau ab:

- **Bachelor-Studierende:** Sie können Kurse nur auf dem Bachelor-Niveau wählen. Master-Kurse sind in der Regel nicht für Bachelor-/Grundstudierende zugänglich. Ausnahmen sind nur möglich, wenn Sie zum Zeitpunkt Ihrer Ankunft im 4. Jahr eines Bachelorstudiums sind.
- **Master-/MBA-Studierende:** Sie können Business-Kurse aus allen Master-Niveau-Kursen wählen. Sollte Ihre Heimatuniversität Ihnen die Teilnahme an ausgewählten Bachelor-Kursen erlauben, sind diese selbstverständlich ebenfalls offen für Sie.

Bei Fragen wenden Sie sich bitte an das Business School Exchange Team unter [incoming.business@uni-mannheim.de](mailto:incoming.business@uni-mannheim.de).

### 📈 Area Finance

**Schwerpunkte:**

- **Finanzprodukte:** Beschreibung des Funktionierens von Finanzmärkten, Identifizierung der Rollen und Interaktionen der Akteure sowie die Nutzung von Finanzmarktdaten zur Implementierung von Finanzstrategien.
- **Anlagenmanagement:** Erfolgreiches Aufbauen und Verwalten eines Portfolios finanzieller Produkte. Identifizierung und Messung finanzieller Risiken sowie die Entscheidungsfindung bei Investitionen.

**Beispiel-Modulstruktur:**

- Session 1: Introduction to Digital Assets as an Alternate Asset Class
- Session 2: Behavioural Biases and IPS
- Session 3: Prepare investment policy statement (IPS) – case study (e-learning)
- Session 4: Valuation of Digital Assets
- Session 5: Fundamental of Programming using Python (e-learning)
- Session 6: Risk and Return of Portfolio with Digital Assets: Theory
- Session 7: Risk and Return of Portfolio with Digital Assets: Practice using Python
- Session 8: Technical Indicators and Trading Strategies: Theory
- Session 9: Technical Analysis: Practice using Python
- Session 10: Capstone Project
