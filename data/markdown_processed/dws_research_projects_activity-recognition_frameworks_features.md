# Sensor Feature Factory

This page provides an overview of the features supported by the [Sensor Feature Factory](https://github.com/sztyler/sensorfeatureextraction) framework.

The framework is designed to extract 10 different types of features, which can be analyzed in both the time- and frequency-domains. A key component of the system is the use of the [JTransforms](https://sites.google.com/site/piotrwendykier/software/jtransforms) library. This library transforms recorded sensor data from the time-domain into the frequency-domain, enabling the computation of advanced metrics, such as expended energy.

## Framework Details
