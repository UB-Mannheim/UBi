# Gratis downloaden und in 5 Minuten an die Servicestelle des DoSV anbinden.

Der Konnektor ist plattformübergreifend nutzbar und läuft auf Windows, Linux oder Mac mit Python ab Version 2.7.9.

### Installation

Die Basisanwendung wird automatisiert mit einem einzigen Aufruf installiert:
`pip install https://konnektor.uni-mannheim.de/downloads/sestco-0.3.zip`

Für eine lückenlose technische Dokumentation ist die \[Sestco Documentation\](Sestco Documentation) verfügbar. Diese erleichtert nicht nur individuelle Anpassungen, sondern sichert auch die sorgfältige Projektpflege in der Zukunft.
