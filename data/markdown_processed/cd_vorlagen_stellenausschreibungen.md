# Stellenausschreibungen

Die Stellenausschreibungen der Universität Mannheim erreichen viele externe Bewerber\*innen. Daher ist es von großer Bedeutung, dass diese Ausschreibungen ein einheitliches Design aufweisen.

Im [Intranet](https://intranet.uni-mannheim.de/arbeitsplatz/personalangelegenheiten/personalgewinnung/stellenanzeigen-generator/) können Sie in wenigen Schritten eine rechtssichere Stellenausschreibung gestalten, die dem Corporate Design der Universität entspricht.
