# Remote Support Tool: Anleitung zur Nutzung

Mit dem Remote Support Tool erhält der IT-Support einen Einblick auf Ihren Bildschirm. Während der Support-Sitzung kann der Support Ihren Bildschirm sehen und Ihre Maus ferngesteuert bewegen. Sie sehen jederzeit, was geschieht, und können gegebenenfalls einschreiten.

## 📺 Videoanleitungen

Vor der Nutzung können Sie sich in unseren Videoanleitungen informieren, wie Sie das Remote Support Tool herunterladen, abspeichern und aufrufen:

## ⚙️ Schritt-für-Schritt-Anleitung

Bitte folgen Sie diesen Schritten, um den Support zu kontaktieren:

### 1. Download des Clients

Laden Sie den entsprechenden Client für Ihr Betriebssystem herunter und speichern Sie die Datei (z.B. auf dem Desktop).

- [Download Anydesk für Windows](https://my.anydesk.com/download/mSOIhmq8/AnyDeskQuickSupport_Win.exe)
- [Download Anydesk für macOS](https://my.anydesk.com/download/7NIwzaKe/AnyDeskQuickSupport_macOS.dmg)

### 2. Start des Tools

Starten Sie das Remote Support Tool „Anydesk“ durch Doppelklick.

- **Hinweis für macOS:** Beim Erstaufruf müssen eventuell einmalig Sicherheitsfreigaben für AnyDesk erteilt werden. Dies ist erkennbar an einem roten Bereich am Fuß des Clients, der die Konfigurationsanforderung anzeigt. Sie werden dabei Schritt-für-Schritt angeleitet.

### 3. Kontaktaufnahme

Bitte nennen Sie dem IT-Support am Telefon Ihre angezeigte ID.

## ⚠️ Wichtige Hinweise

- **Keine dauerhafte Installation:** Es werden keine Komponenten des Programms dauerhaft auf Ihrem System installiert.
- **Transparenz:** Sie sehen jederzeit live, was auf Ihrem System geschieht.
- **Ende der Sitzung:** Nach Beendigung der Support-Sitzung hat der IT-Support keinen Zugriff mehr auf Ihr System.
