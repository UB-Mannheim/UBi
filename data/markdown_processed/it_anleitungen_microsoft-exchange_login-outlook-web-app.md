# Login in die Outlook Web App (OWA)

Diese Anleitung zeigt Ihnen, wie Sie Ihr Exchange-Postfach online über den Webmailer OWA (Outlook Web App) aufrufen.

**Anleitung:**

1. Rufen Sie die [Outlook Web App (OWA)](https://exchange.uni-mannheim.de/) auf.
1. Geben Sie unter „Benutzername“ Ihre Uni-ID und unter „Kennwort“ das zu Ihrer Uni-ID gehörige Passwort ein.
1. Klicken Sie anschließend auf „Anmelden“.

Sie gelangen nun zu Ihrem Postfach.
