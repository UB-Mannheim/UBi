# Information Retrieval and Web Search (IE663)

Given the vastness and richness of the Web, users require high-performing, scalable, and efficient methods to access information. This MSc course explores a variety of basic and advanced techniques for text-based information retrieval and Web search.

## Course Overview

**IE 663: Information Retrieval and Web Search**

- **Focus:** Learning techniques for searching and effectively retrieving relevant pieces of information from large text collections.
- **Format:** Coursework includes exercises and a final exam (IE 663). Homework assignments are designed to prepare students for the final exam.
- **Related Course:** This course is related to the **IR Project (IE 691)**, which is a separate, team-based project (teams of 4–5 people). The project focuses on various IR problems covered in class and requires deliverables including software (code and documentation) and a short evaluation report.

## Prerequisites

Students should have a solid foundation in:

- Fundamental notions of linear algebra.
- Probability theory.
- Algorithms and data structures.
- Programming skills (Java or Python are recommended for the IE 691 Project).
