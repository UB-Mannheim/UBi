# IE 675b: Machine Learning (HWS 2025)

This document provides details for the course *IE 675b: Machine Learning*, designed for Master candidates.

## 📚 Kursübersicht (Organization)

- **Thema:** Machine Learning
- **Credits:** 9 ECTS points
- **Format:** Lecture, exercises, assignments
- **Voraussetzungen:** IE 500 Data Mining I (recommended / in parallel), knowledge of probability and statistics.
- **Anmeldung:** Enrollment is required via Portal 2.

### 🧑‍🏫 Personal

- **Dozent (Lecturer):** [Prof. Dr. Rainer Gemulla](https://www.uni-mannheim.de/dws/people/professors/prof-dr-rainer-gemulla/)
- **Tutor:** [Julie Naegelen](https://www.uni-mannheim.de/dws/people/researchers/phd-students/julie-naegelen/)

### 🗓️ Termine

- **Format:** The lecture (2 sessions per week) and tutorial (1 session per week) will be held in-presence.
- **Start:** Both components begin in the first week.
- **Kickoff:** Details are discussed in the **kickoff lecture (Tuesday, Sep 2, 10:15)**.

## 🧠 Kursinhalt (Content)

Machine learning involves building computer systems that improve with experience, alongside the study of learning processes. The module provides an introduction to the field, covering underlying concepts, theoretical principles, and algorithms.

**The curriculum covers:**

- Basics of machine learning
- Training, inference, and decision processes
- **Generative Models:** (e.g., Beta-binomial, Naive Bayes, PPCA)
- **Discriminative Models:** (e.g., logistic regression, SVM, GPR)
- **Latent Variable Models:** (e.g., SVD, LLM, GMM)
- The EM algorithm
- Kernel methods
- Hyperparameter optimization

> **Hinweis:** Deep learning and more advanced topics are covered in the follow-up course “IE 678 Deep Learning” (and others).
