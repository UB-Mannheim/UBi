# E-Mail aus einem Funktionspostfach im Webmailer OWA erstellen

Diese Anleitung beschreibt, wie Sie eine E-Mail aus einem Funktionspostfach (Shared Mailbox) mithilfe der Outlook Web App (OWA) verfassen.

### 📧 Schritt-für-Schritt-Anleitung

1. **Zugriff auf OWA:** Rufen Sie die Outlook Web App (OWA) unter [https://exchange.uni-mannheim.de](https://exchange.uni-mannheim.de/) auf.
1. **Anmelden:** Geben Sie Ihre Uni-ID und Ihr zugehöriges Passwort ein und klicken Sie auf „Anmelden“.
1. **Weiteres Postfach öffnen:** Klicken Sie oben rechts auf Ihr Bild/Ihre Initialen und wählen Sie „Weiteres Postfach öffnen“.
1. **Funktionspostfach auswählen:** Geben Sie im erscheinenden Fenster den Namen des Funktionspostfachs ein und klicken Sie auf „Öffnen“.
   - *Hinweis:* Das Funktionspostfach öffnet sich in einem neuen Tab. Falls Ihr Browser eine Warnmeldung anzeigt, erlauben Sie bitte das Öffnen von Tabs für [exchange.uni-mannheim.de](https://exchange.uni-mannheim.de/).
1. **E-Mail verfassen:** Klicken Sie auf **Neu → E-Mail**.
   - Die E-Mail wird nun mit der E-Mail-Adresse des Funktionspostfachs als Absender\*in verfasst.
1. **Abschluss:** Zum Beenden der Arbeit schließen Sie einfach den Tab des Funktionspostfachs. Sie kehren dabei zu Ihrem eigenen Postfach im Webmailer zurück.
