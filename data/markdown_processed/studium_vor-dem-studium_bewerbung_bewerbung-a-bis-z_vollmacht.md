# Vollmacht bei der Bewerbung

### Bewerbung ohne Unterschrift der Eltern

Sie können sich für einen Studienplatz bewerben, auch wenn Sie noch nicht volljährig sind. Hierfür benötigen Sie weder die Unterschrift Ihrer Eltern noch eine Vollmacht.

### Wann ist eine Vollmacht notwendig?

Eine Vollmacht ist ausschließlich dann erforderlich, wenn Sie wünschen, dass Sie von einer dritten Person vertreten werden.

**Wichtig:**

- Ohne eine vorliegende Vollmacht können Dritte (wie beispielsweise Ihre Eltern) weder Informationen zu Ihrer Bewerbung erhalten, noch Ihre Einschreibung vornehmen.
- Falls Sie eine Vertretung wünschen, reichen Sie die Vollmacht zusammen mit den restlichen Bewerbungsunterlagen ein.
