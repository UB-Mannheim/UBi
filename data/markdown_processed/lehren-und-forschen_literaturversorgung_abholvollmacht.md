# Abholvollmacht und UB-ID für wissenschaftliche Beschäftigte

## Abholvollmacht

Hauptamtliche wissenschaftliche Beschäftigte der Universität Mannheim können eine Abholvollmacht für ihr Bibliothekskonto ausstellen lassen. Mit der Abholvollmacht berechtigen Sie andere Personen, Medien über Ihr Bibliothekskonto auszuleihen. Die Abholvollmacht muss persönlich mit Ihrer ecUM im [InfoCenter](https://www.bib.uni-mannheim.de/standorte/infocenter/) abgeholt werden. Zum Ausleihen von Medien ist die Vorlage der Originalvollmacht erforderlich.

Die Abholvollmacht berechtigt nicht zum Zugriff auf das Bibliothekskonto. Wenn Sie anderen Personen Zugriff auf Ihr Bibliothekskonto ermöglichen möchten, beantragen Sie eine Zusatz-ID.

## UB-ID

Lehrstuhlinhaberinnen und Lehrstuhlinhaber können eine zusätzliche UB-ID beantragen, die ausschließlich den Zugriff auf das eigene Bibliothekskonto erlaubt. Diese UB-ID kann an Beschäftigte weitergegeben werden, um Bestellungen, Vormerkungen und Verlängerungen vorzunehmen. Die zusätzliche ID erhalten Sie über die [Universitäts-IT](https://www.uni-mannheim.de/it/services/) im InfoCenter.
