# Automatisierte Texterkennung: Datenerhebung via OCR/HTR

Die maschinelle Texterkennung (OCR) ermöglicht die automatisierte Erfassung von Texten aus digitalen Bildern, wodurch durchsuchbare und analysierbare Daten entstehen. Die Universitätsbibliothek Mannheim bietet umfassende Unterstützung für Forschende der Universität Mannheim, von der Digitalisierung über die Layout- und Texterkennung bis hin zur Strukturierung der Daten.

## Texterkennungs- und Transkriptionsplattformen

Hier ist eine Auswahl an Texterkennungs- und Transkriptionsplattformen:

| Tool               | Kostenmodell            | Eigenschaften                                         | Besonders geeignet für                          |
|--------------------|------------------------|------------------------------------------------------|------------------------------------------------|
| ABBYY Finereader   | kostenpflichtig        | Text- und Layouterkennung; gute Layoutanalyse        | Moderne Drucke, komplexes Layout                |
| eScriptorium       | Open Source            | Intuitive Nutzung, GUI für Kraken                     | Historische Drucke und Handschriften            |
| Google Vision      | kostenpflichtig        | Texterkennung; Bild- und Videoanalyse                | Drucke und Handschriften                        |
| Kraken             | Open Source            | Kommandozeilenbasierte Software                       | Historische Drucke und Handschriften            |
| OCR4All            | Open Source            | GUI für verschiedene Texterkennungsprogramme         | Historische Drucke und Handschriften            |
| OCRmyPDF           | Open Source            | Texterkennung von PDF-Dateien                         | Historische/moderne Drucke                      |
| OCR-D              | Open Source            | Modular aufgebaute Texterkennungssoftware            | Historische Drucke                              |
| PERO-OCR           | Open Source            | Webbasierte Plattform                                 | Historische/moderne Drucke und Handschriften    |
| Tesseract          | Open Source            | Kommandozeilenbasierte Software                       | Historische/moderne Drucke                      |
| Transkribus        | kostenpflichtig        | Umfassende Texterkennungs- und Transkriptionsplattform| Historische Handschriften und Tabellen          |
