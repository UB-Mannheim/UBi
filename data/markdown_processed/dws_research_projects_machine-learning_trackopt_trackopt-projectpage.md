# TrackOpt: Learning to Optimize Physically Constrained Sparse-to-Dense Point Tracking

The TrackOpt project develops an advanced, resilient framework for tracking individually occurring points or dense point clouds. This technology is highly relevant across various physical and biological domains, where tracking methods must function despite limited annotated training data and inherent measurement uncertainties.

## Project Goal and Methodology

The primary aim is to create an efficient and flexible method framework that can reliably solve new tracking problems. The core methodology involves integrating model-driven, potentially discrete optimization problems with deep neural networks.

By combining these two approaches, the framework achieves:

1. **Data Efficiency:** The integration of model-driven knowledge allows the system to incorporate physical or biological constraints, making it highly data-efficient.
1. **Resilience:** Learned networks are optimized to predict consistent solutions despite measurement inaccuracies and uncertainty.

The framework is designed to be resilient and flexible, enabling the integration of current tracking problems across three key application fields:

- Particle Physics
- Microfluidics
- Microscopy

## Software Toolbox and Availability

The project will develop a reusable, transdisciplinary, and modular software toolbox. This open-source application will be made available to the public, allowing external parties in various research and application areas to independently solve complex tracking problems with different boundary conditions.
