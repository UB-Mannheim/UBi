# Platform Features and User Roles

The features available on the platform are dependent on the user's assigned role. Different roles have access to different functionalities, which are structured by the user journey and the user's role.

### User Journey Phases

The platform is designed to guide users through various stages. The features are structured by role and by the chronological appearance along the user journey:

1. **Orientation**
1. **Application**
1. **Education** (Note: This phase does not take place within the platform.)
1. **Certification** (Note: Features related to this stage are currently under development and will be introduced in future updates.)
