# David Jehn

David Jehn ist ein Stipendiat des Spitzensport-Stipendiums der Metropolregion Rhein-Neckar.

### Studiendetails

| Kriterium | Details |
| :--- | :--- |
| **Förderungsbeginn** | HWS 2021/2022 |
| **Sportart** | Feldhockey |
| **Studium** | Unternehmensjurist |
| **Verein** | TSV Mannheim Hockey |

### Sportliche Erfolge

Seine größten sportlichen Erfolge umfassen:

- 3. Platz Sommerturnier U18 (2020)
- 1. Platz Länderpokal Hamburger Auswahl (2019)
- 2. Platz Deutsche Meisterschaft
- 1. Platz Nord-Ost Deutsche Meisterschaft
