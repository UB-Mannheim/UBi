# Vorlesung

Eine Vorlesung bezeichnet eine spezifische Lehrveranstaltungsform, die typischerweise von Professor\*innen oder promovierten Dozierenden gehalten wird.

**Format und Dauer:**

- **Charakter:** Die Vorlesung hat den Charakter eines Vortrages.
- **Dauer:** Sie dauert in der Regel 90 Minuten.

**Besondere Formate:**

- **Ringvorlesungen:** Hierbei halten abwechselnd verschiedene Professor\*innen oder Dozierende die wöchentlichen Vorträge.

Für eine allgemeine Übersicht über das Studium und das Verstehen des Studierens finden Sie weitere Informationen unter [Studieren verstehen](https://www.uni-mannheim.de/studium/studieren-von-a-bis-z/).
