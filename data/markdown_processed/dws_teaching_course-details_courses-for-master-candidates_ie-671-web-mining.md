# Web Mining Course Overview (FSS2025)

The Web provides vast amounts of textual and structured data, making it a rich source for knowledge extraction. This course teaches techniques for mining Web content to derive knowledge for business decisions and applications.

## Course Components

The course is divided into two parts: a lecture and accompanying practical projects.

### 📚 Lecture (IE671)

- **Focus:** Theory and methods of web mining.
- **Credits:** 3 ECTS.
- **Topics Covered:**
  - Web Usage Mining (including Recommender Systems)
  - Web Structure Mining (including Social Network Analysis)
  - Web Content Mining (including Sentiment Analysis and Hate Speech Detection)

### 🛠️ Project Work (IE684)

- **Focus:** Practical application of web mining methods.
- **Format:** Students work in teams, applying methods to a real-world use case.
- **Assessment:** Results are reported via a written report and an oral presentation.
- **Credits:** 3 ECTS.

> **Recommendation:** While the lecture and project can be attended in separate years, it is highly recommended to attend both in the same semester as the schedules are aligned.
