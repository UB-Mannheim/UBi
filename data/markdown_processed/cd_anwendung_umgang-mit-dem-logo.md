# Umgang mit dem Logo

Das Universitätslogo ist das Key Visual der Marke Universität Mannheim und darf nicht verändert werden. Dies schließt auch die Nutzung von einzelnen Elementen des Logos mit ein. Benutzen Sie deshalb nur die Logos, die auf der [Website](https://www.uni-mannheim.de/cd/marke/logos/) zur Verfügung stehen.

Die Verwendung des Logos ist der Universität Mannheim vorbehalten. Es darf daher nur in klar definierten Anwendungsfeldern durch die Universität eingesetzt werden. Erläuterungen, welche internen Gruppen das Logo nutzen können, finden Sie [hier](https://www.uni-mannheim.de/cd/anwendung/umgang-mit-dem-logo/verwendung-des-logos/). Für die Nutzung durch externe Gruppen ist eine vorherige Genehmigung erforderlich.
