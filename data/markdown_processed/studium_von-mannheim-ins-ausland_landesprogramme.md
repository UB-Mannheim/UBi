# Landesprogramme: Studium im Ausland

Landesprogramme sind multilaterale Kooperationen zwischen den baden-württembergischen Hochschulen und einem Verbund öffentlicher Hochschulen eines amerikanischen Bundesstaates (z. B. das California State University System oder das North Carolina State University System).

Auf baden-württembergischer Seite werden diese Programme vom Ministerium für Wissenschaft, Forschung und Kunst (MWK) koordiniert und mitfinanziert. Da an einem Austausch über ein Landesprogramm mehrere baden-württembergische Hochschulen beteiligt sind, ist eine genaue Absprache und Koordination zwischen den verschiedenen Campus erforderlich. Für jedes Landesprogramm gibt es eine koordinierende Universität auf baden-württembergischer und eine auf nordamerikanischer Seite.

## Bewerbungsverfahren

Die Bewerbung für Landesprogramme erfolgt in einem zweistufigen Prozess:

### 1. Grundsätzliche Landesprogramm-Bewerbung und Zusage

Zunächst bewerben Sie sich generell an der Universität Mannheim um die Teilnahme an einem Landesprogramm. Die Bewerbung erfolgt wie für die regulären [Auslandsbewerbungsterminen](https://www.uni-mannheim.de/studium/von-mannheim-ins-ausland/austauschstudium/bewerbung/bewerbungsanleitungen/) der Universität Mannheim.

- **Wunschziele:** Sie können mehrere Wunschziele aus Regionen angeben, die zu diesem Zeitpunkt ausgeschrieben sind. Sie können Partneruniversitäten und Landesprogramme in einer Bewerbung mischen.
- **Doppelte Nennung:** Eine Universität kann sowohl direkter Austauschpartner als auch Teilnehmer an einem Landesprogramm sein. In diesem Fall können Sie die gewünschte Universität in Ihrer Bewerbung zweimal angeben.

### 2. Platzierung innerhalb des Landesprogramms

Nach Erhalt einer **Zusage für die Teilnahme** erfolgt in einem zweiten Schritt die **Platzierung an einer Universität** innerhalb des Landesprogramms.

- **Ablauf:** Sie nennen Ihre bevorzugten Universitäten. Über die finale Platzierung entscheidet eine Kommission des Landesprogramms zusammen mit den ausländischen Universitäten.
- **Sicherheit:** Die Kommission wird versuchen, Sie an einer Ihrer Wunschuniversitäten zu platzieren. Sollte dies nicht möglich sein, wird Ihnen aber auf jeden Fall ein geeigneter Platz für Ihr Studienfach an einer anderen Universität angeboten, wodurch Ihre Teilnahme gesichert ist.
