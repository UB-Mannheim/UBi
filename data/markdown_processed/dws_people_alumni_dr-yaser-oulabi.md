## 🔬 Forschungsschwerpunkte (Research Focus)

Dr. Oulabi war ehemaliger Forscher am [Chair of Information Systems V: Web-based Systems](https://www.uni-mannheim.de/dws/research/focus-groups/web-based-systems-prof-bizer/) von [Prof. Dr. Christian Bizer](https://www.uni-mannheim.de/dws/people/professors/prof-dr-christian-bizer/). Seine Forschung konzentriert sich auf die Erweiterung von Domänen-übergreifenden Wissensdatenbanken mithilfe strukturierter Webdaten.

**Kernbereiche:**

- **Knowledge Base Completion:** Vervollständigung von Wissensdatenbanken.
- **Web Table Data Consolidation:** Konsolidierung von Daten aus Web-Tabellen.
- **Long-Tail Entity Extraction:** Extraktion von seltenen (Long-Tail) Entitäten.
- **Time-Dependent Web Data Fusion:** Fusion von zeitabhängigen Webdaten.
- **Weak Supervision:** Einsatz von schwacher Überwachung zur Datenverarbeitung.

## 💾 Veröffentlichte Datensätze (Published Datasets)

Diese Datensätze dienen als Goldstandard für die Entwicklung und Evaluierung von Methoden zur Datenfusion und Entitätsextraktion.

- **[Time-Dependent Ground-Truth Dataset](http://webdatacommons.org/TDGT/)**
  - Ground truth von zeitabhängigen Daten aus verschiedenen Domänen. Ermöglicht die Entwicklung und Evaluierung von Methoden für zeitabhängige Daten.
- **[Web Tables for Long-Tail Entity Extraction](http://webdatacommons.org/T4LTE/)**
  - Goldstandard zur Evaluierung der Extraktion von Long-Tail-Entitäten aus Web-Tabellen. Entwickelt zur Erweiterung von Wissensdatenbanken mit bisher unbekannten Entitäten.
