# Hugo von Montgelas

Hugo von Montgelas ist ein Stipendiat des Spitzensport-Stipendiums der Region Rhein-Neckar.

## Studiendetails

| Kriterium | Details |
| :--- | :--- |
| **Förderungsbeginn** | HWS 2022/2023 |
| **Studium** | Wirtschaftspädagogik |
| **Sportart** | Feldhockey |
| **Verein** | Mannheimer HC |

## Sportliche Erfolge

Seine größten sportlichen Erfolge umfassen:

- Debüt in der A Nationalmannschaft (2022)
- 1. Platz Europameisterschaft U19 (2021)
