# Angaben zu Ihrer Erstinschreibung

Die Onlinebewerbung erfordert die Angabe Ihrer bisherigen akademischen Laufbahn. Bitte beachten Sie die spezifischen Richtlinien für die Eingabe der Daten.

## 🎓 Erforderliche Angaben zur Studienhistorie

Die folgenden Felder müssen ausgefüllt werden:

- **Wann haben Sie zum ersten Mal studiert?**
  - Geben Sie das Semester an, in dem Sie zum ersten Mal eingeschrieben waren – unabhängig vom Land der Erstimmatrikulation.
  - *Falls Sie noch nie studiert haben:* Geben Sie das Semester an, in das Sie sich aktuell bewerben.
- **Land der Ersthochschule**
  - Wählen Sie das Land Ihrer Ersthochschule aus.
  - *Falls Sie noch an keiner Hochschule eingeschrieben waren oder sind:* Geben Sie Deutschland an.
- **Erstsemester in Deutschland**
  - Geben Sie das Semester an, in dem Sie zum ersten Mal an einer deutschen Hochschule eingeschrieben waren.
  - *Falls Sie noch nie studiert haben:* Geben Sie das Semester an, in das Sie sich aktuell bewerben.
- **Ersthochschule in Deutschland**
  - Nennen Sie die Hochschule, an der Sie erstmals in Deutschland eingeschrieben waren (unabhängig vom Studiengang).
  - *Falls Sie noch an keiner deutschen Hochschule eingeschrieben waren oder sind:* Geben Sie die **Universität Mannheim** an.
- **Bisherige Hochschulsemester**
  - Hochschulsemester umfassen *alle Semester an deutschen Hochschulen* (einschließlich Urlaubs- und Fachsemestern anderer Studiengänge). Die genaue Anzahl ist in den aktuellen Studienpapieren Ihrer letzten Hochschule vermerkt.
  - *Falls Sie noch an keiner deutschen Hochschule eingeschrieben waren oder sind:* Geben Sie den Wert **„0“** an.
- **Erstsemester an der Universität Mannheim**
  - Geben Sie das Semester an, in dem Sie zum ersten Mal an der Universität Mannheim eingeschrieben waren.
  - *Falls Sie bis dato noch nie an der Universität Mannheim eingeschrieben waren:* Geben Sie das Semester an, in das Sie sich aktuell bewerben.

## 📝 Wichtige Hinweise zur Bewerbung

- **Semesterbegrenzung:** Die Onlinebewerbung erlaubt die Angabe von maximal 15 Hochschulsemestern. Sollten Sie mehr Semester vorweisen können, geben Sie bitte trotzdem 15 Semester an.
- **Datenkorrektur:** Die eingereichten Daten werden später anhand der von Ihnen hochgeladenen Belege in der Datenbank korrigiert.
