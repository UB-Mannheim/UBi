# Image Processing (CS-647)

This course provides a comprehensive introduction to the fundamentals of image processing. The curriculum starts with the human visual system and the basics of digital image acquisition. The primary goal is to provide students with a deep understanding of the technical and theoretical basis of image processing, enabling them to implement basic algorithms in practice.
