# E-Mail- und Kalender-Service mit Exchange

Die Universität Mannheim stellt allen Beschäftigten und Studierenden einen eigenen E-Mail-Account mit Postfach im **Exchange Mailsystem** zur Verfügung. Dieses System ermöglicht die Verwaltung von E-Mails, Kalendern und Kontakten.

## 📧 Zugriff auf das Exchange-Postfach

Sie haben zwei Hauptmöglichkeiten, auf Ihr Exchange-Postfach zuzugreifen:

### 1. Über den Webmailer OWA (Outlook Web App)

Rufen Sie die Seite [exchange.uni-mannheim.de](https://exchange.uni-mannheim.de) auf.

- **Login-Daten:** Sie benötigen Ihre Uni-ID und das dazugehörige Passwort.
- *Passwort vergessen?* [Hier finden Sie die Anleitung](https://www.uni-mannheim.de/it/anleitungen/passwort/).

### 2. Über einen E-Mail-Client

Für die Verwaltung von E-Mails, Terminen und Adressen können Sie auch dedizierte Programme nutzen. Wir bieten Schritt-für-Schritt-Anleitungen für verschiedene Betriebssysteme:

- [Anleitung für Windows-PC/ -Laptop (Outlook, Thunderbird oder Windows Mail App)](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/einbinden-windows/)
- [Anleitung für MacBook oder iMac (Outlook oder Apple Mail)](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/einbinden-macos/)
- [Anleitung für iPhone oder iPad](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/einbinden-ios/)
- [Anleitung für Android Smartphone oder Tablet](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/einbinden-android/)
- [Anleitung für Linux-PC/ -Laptop (Thunderbird)](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/einbinden-linux/)

## ✨ Funktionen des E-Mail-Services

Das stabile Mailsystem bietet Ihnen zahlreiche Vorteile, darunter:

- **Speicherplatz:** 6 GB Speicherplatz.
- **Sicherheit:** Umfassender SPAM-Schutz.
- **Backup & Wiederherstellung:** Tägliche lokale Sicherung der Daten im Netz der Universität Mannheim. Daten können bis zu zwei Wochen nach Löschen wiederhergestellt werden (siehe [Anleitung zur Wiederherstellung gelöschter Mails](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/wiederherstellen-geloeschter-mails/)).
- **Organisation:** Verwaltung von Regeln zur Organisation eingehender Nachrichten.
- **Kontakte:** Globale Adressbücher für schnelles Auffinden von Mailadressen.
- **Kalender:** Freigabe von Kalendern inklusive individueller Vergabe von Zugriffs- und Bearbeitungsrechten.
- **Terminplanung:** Komfortable Terminplanungsassistenten, die die Verfügbarkeit der Teilnehmenden anzeigen.
- **Abwesenheitsnachrichten:** Aktivierung automatischer Antworten (Abwesenheitsnachrichten) für definierte Zeiträume.
- **Massenkommunikation:** Erstellung von Mailinglisten für den Versand von Serien- bzw. Massenmails über [Mailman](https://www.uni-mannheim.de/it/services/digitale-kommunikation/mailman/).

> **Wichtiger Hinweis zu Weiterleitungen:** Sie können eingehende Nachrichten auf eine andere universitäre E-Mail-Adresse automatisch weiterleiten (siehe [Anleitung Weiterleitung](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/weiterleitung-einrichten/)). **Achtung:** Die Weiterleitung an private Mailkonten oder universitätsexterne Server ist allen Beschäftigten der Universität Mannheim nicht gestattet.

## ⚙️ Technische Details und Voraussetzungen

### Nutzungsvoraussetzungen

- **Rechtlich:** Der Nutzer ist selbst verantwortlich für die Freigabe seiner Adressbücher oder Kalender sowie gespeicherte Dateninhalte. Bitte beachten Sie die [Benutzungsordnung](https://www.uni-mannheim.de/media/Einrichtungen/it/Benutzerordnung_und_wichtige_Dokumente/Benutzungsordnung_Informationssysteme_Universitaet_Mannheim.pdf).
- **Technisch:** Alle gängigen Webbrowser werden unterstützt. Für die vollumfängliche Nutzung wird die Verwendung von Microsoft Exchange über die Webanwendung oder den E-Mail-Client Outlook empfohlen.
- **Zugang:** Sie benötigen Ihre Uni-ID sowie das zugehörige Passwort.

### IMAP/SMTP-Einstellungen für Exchange

Wir empfehlen die Verwendung von „Exchange“ als Kontotyp/Protokoll. Falls Sie eine Verbindung über IMAP herstellen möchten, verwenden Sie folgende Daten:

**IMAP (Posteingangsserver):**
| Protokoll | IMAP |
| :--- | :--- |
| Server | exchange.uni-mannheim.de |
| Port | 993 |
| Sicherheitstyp/Verschlüsselung | SSL/TLS |
| Benutzername | Uni-ID |
| Passwort | Passwort Ihrer Uni-ID |

**SMTP (Postausgangsserver):**
| Server | exchange.uni-mannheim.de |
| :--- | :--- |
| Port | 587 |
| Sicherheitstyp/Verschlüsselung | STARTTLS |
| Anmeldung/Authentifizierung erforderlich | ja |
| Benutzername | Uni-ID |
| Passwort | Passwort Ihrer Uni-ID |
