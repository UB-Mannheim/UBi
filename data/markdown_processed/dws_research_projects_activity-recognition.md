# Human Activity Recognition (HAR)

This project provides tools, datasets, and results for the automatic detection of human activity recognition (e.g., walking) and activities of daily living (e.g., watering plants).

## Applications

The goal is to develop robust activity recognition methods using mobile device sensors that yield high-quality results in real-world settings. This section provides developed apps, including source code and documentation. We offer a comprehensive data collector app and are currently developing a HAR app for everyday life.

## Datasets

The dataset covers acceleration, GPS, gyroscope, light, magnetic field, and sound level data collected from fifteen subjects performing various activities. These activities include climbing stairs (up and down), jumping, lying, standing, sitting, running/jogging, and walking. For each activity, the on-body positions of the chest, forearm, head, shin, thigh, upper arm, and waist were simultaneously recorded.

## Frameworks

We have developed useful frameworks for processing the recorded sensor data. These frameworks are available for free use and can be modified or enhanced without further requests. Currently, our focus is on deriving useful information from accelerometer data, specifically by segmenting the data stream and computing meaningful features.
