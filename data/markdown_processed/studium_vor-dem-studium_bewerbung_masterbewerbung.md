# Masterbewerbung an der Universität Mannheim

Für die Masterstudiengänge an der Universität Mannheim benötigen Sie einen **ersten Studienabschluss** sowie bestimmte **fachliche Voraussetzungen**, die sich nach Ihrem gewünschten Studienfach richten.

## Bewerbungsverfahren

Die Bewerbung hängt davon ab, ob Sie in das erste oder ein höheres Fachsemester einsteigen möchten:

- **[Bewerbung in das erste Fachsemester](https://www.uni-mannheim.de/studium/vor-dem-studium/bewerbung/masterbewerbung/erstes-fachsemester-master/)**
  Hier finden Sie alle Informationen zur Bewerbung für das erste Master-Fachsemester.
- **[Bewerbung in ein höheres Fachsemester](https://www.uni-mannheim.de/studium/vor-dem-studium/bewerbung/masterbewerbung/hoeheres-fachsemester-master/)**
  Dieser Link richtet sich an Studierende, die bereits studiert haben und zu uns wechseln möchten.

### Wichtige Informationsquellen

- **[Auswahlsatzungen](https://www.uni-mannheim.de/studium/vor-dem-studium/bewerbung/bewerbung-a-bis-z/auswahlsatzungen/)**
  In den Auswahlsatzungen sind die spezifischen Zugangsvoraussetzungen und Auswahlkriterien der einzelnen Masterstudiengänge festgelegt.
- **[Bewerbung von A-Z](https://www.uni-mannheim.de/studium/vor-dem-studium/bewerbung/bewerbung-a-bis-z/)**
  Hier finden Sie wichtige Informationen zum gesamten Bewerbungsprozess, von der "Amtlichen Beglaubigung" bis zur "Zulassung".

**Tipp:** Wenn Sie wissen, dass Sie im laufenden Semester Ihr Bachelorstudium abschließen werden, können Sie sich bereits für den Master bewerben. Details dazu entnehmen Sie bitte den jeweiligen [Auswahlsatzungen](https://www.uni-mannheim.de/studium/vor-dem-studium/bewerbung/bewerbung-a-bis-z/auswahlsatzungen/).

*Interesse an einem Masterstudium?* Abonnieren Sie unseren [Reminder für die Bewerbungsfrist Masterstudiengänge](https://www.uni-mannheim.de/studium/termine/bewerbungsfristen/reminder-bewerbungsfrist-master/)!
