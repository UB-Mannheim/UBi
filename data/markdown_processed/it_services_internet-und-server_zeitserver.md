# Zeitserver (NTP-Server)

Die Universitäts-IT betreibt einen zentralen Zeitserver (auch „Timeserver“ oder „NTP-Server“). Dieser dient als Referenzquelle und ermöglicht die Synchronisierung einer genauen Systemzeit von Computer-Systemen über das **NTP-Protokoll**. Eine präzise Zeitsynchronisierung ist für ein reibungsloses Zusammenspiel von Systemen und Anwendungen unerlässlich.

### Erreichbarkeit

Der zentrale Zeitserver ist innerhalb der Mannheimer Universitätsnetze unter folgenden Adressen erreichbar und benutzbar:

- **IP-Adresse:** `134.155.106.19`
- **Hostname:** `rumtime.uni-mannheim.de`
