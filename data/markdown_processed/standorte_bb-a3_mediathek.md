# Mediathek der Universität Mannheim

Die Mediathek im Bibliotheksbereich A3 bietet eine umfangreiche Sammlung von Filmen auf DVD. Hier können Sie Filme in einem speziell ausgestatteten Arbeitsbereich ansehen.

## Filme auf DVD

- DVDs können an speziell ausgestatteten Arbeitsplätzen in A3 angesehen werden.
- Hauptamtliches wissenschaftliches Personal, Promotionsstudierende sowie Gastwissenschaftlerinnen und Gastwissenschaftler können DVDs für ein Semester ausleihen.
- Studierende der Universität Mannheim haben die Möglichkeit, DVDs während der Hausarbeits- und Prüfungs­vorbereitungszeit auszuleihen, sofern sie eine [Dozentenbescheinigung](https://www.bib.uni-mannheim.de/media/Einrichtungen/Universitaetsbibliothek/Dokumente/Medien/Dozentenbescheinigung_Mediathek_Ausleihe_2023.doc) (PDF, 285 KB) vorlegen.

Die Mediathek ist Mitglied im [Netzwerk Mediatheken](https://www.netzwerk-mediatheken.de/). Um den gesamten Bestand der Filme auf DVD zu durchsuchen, nutzen Sie die **„Erweiterte Suche“** und geben Sie die **Zweigstelle „medio“** ein.
