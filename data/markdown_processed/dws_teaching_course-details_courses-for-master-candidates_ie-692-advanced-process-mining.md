# IE 692: Advanced Process Mining (FSS 2024)

Process mining is an emerging branch of data science that aims at deriving qualitative and quantitative insights on the execution of organizational processes, based on the analysis of recorded event sequences.

This course features lectures and exercises that focus on the formal foundations, algorithms, and techniques of process mining.

## Covered Topics

The course covers fundamental algorithms as well as advanced, state-of-the-art techniques in the following areas:

- **Process Discovery:** Deriving a process model from recorded events.
- **Conformance Checking:** Identifying deviations between event data and established process models.
- **Process Enhancement:** Augmenting process models with information regarding the temporal, organizational, and data perspectives of a process.
- **Predictive Monitoring:** Making predictions about ongoing process instances.
- **Data Preprocessing:** Techniques to preprocess, abstract, and cluster event data for improved analyses.

## Learning Format and Goals

### Course Structure

The learning process combines:

1. **Lectures:** Theoretical foundations and advanced techniques.
1. **Exercises:** Practical application through pen-and-paper exercises, implementation, and evaluation using open-source process mining tools and libraries.
1. **Group Assignment:** A practical project where students implement and/or evaluate a process mining technique.

### Learning Outcomes

After completing this course, students will be able to:

- Understand the importance and potential of process mining.
- Know and apply both fundamental and advanced techniques for core process mining tasks.
- Analyze real-world data using open-source process mining tools.
