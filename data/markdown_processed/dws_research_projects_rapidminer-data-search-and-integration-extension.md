# RapidMiner Data Search and Integration Extension

The Data Search and Integration Extension is an extension designed for the open-source data mining software [RapidMiner](http://rapid-i.com/content/view/181/190/).

## Purpose and Functionality

The primary goal of the extension is to support the iterative extension of data tables. It assists users in the process of finding appropriate records necessary for their data mining tasks.

## Project History

A first demonstration of the extension was proposed at the [ESWC2016](https://www.uni-mannheim.de/dws/research/projects/extending-rapidminer-with-data-search-and-integration-capabilities/) conference.
