# 🖨️ Ricoh Kopierer als Drucker unter Windows einrichten

Diese Anleitung beschreibt, wie Sie einen Ricoh Kopierer als Drucker auf Ihrem Windows-System einrichten.

**⚠️ Wichtiger Hinweis vorab:**

- Diese Anleitung gilt **nicht** für die Verwaltung oder Konfiguration des Kopierers selbst.
- Für administrative Aufgaben wenden Sie sich bitte direkt an den **IT-Support**:
  - **Hotline:** -2000
  - **E-Mail:** itsupport@uni-mannheim.de
  - **MS Teams:** „UNIT Support“

### 🌐 Wichtige Hinweise zum Netzwerkzugriff

Bitte beachten Sie die folgenden Einschränkungen bezüglich des WLAN-Zugriffs:

- **Mitarbeitende (Staff):** Kopierer aus dem Universitäts-WLAN „eduroam“ können nur angesprochen werden, wenn das Endgerät mit einer Uni-ID angemeldet ist, die die Ausprägung **„staff“** hat.
- **Studenten:** Studentische Kennungen (Ausprägung „student“) können aus Sicherheitsgründen leider nicht direkt auf die Kopierer zugreifen.
