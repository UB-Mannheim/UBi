# Extending RapidMiner with Data Search and Integration Capabilities

This project addresses the challenge faced by analysts who need data for a data mining project but struggle to locate it, whether it resides on the Web or within an organization's intranet. While existing data mining tools offer powerful methods, they often lack robust support for searching for suitable data and integrating information from multiple sources.

### Project Overview

The research team developed an extension for **[RapidMiner](https://rapidminer.com/)**, a popular data mining framework. This extension enables analysts to:

1. **Search for relevant datasets** across various sources.
1. **Integrate discovered data** with existing datasets.

A key feature supported is the iterative extension of data tables with additional attributes.

### Achievements and Recognition

The demonstration of this extension was presented at the **13th European Semantic Web Conference (ESWC2016)**. The demo was one of the 19 accepted presentations and notably won the **best demonstration award**.

### Research Context

The RapidMiner Extension is developed as part of the BMBF-funded research project:

- [DS4DM: Data Search for Data Mining](https://www.uni-mannheim.de/dws/research/projects/ds4dm-data-search-for-data-mining/)
