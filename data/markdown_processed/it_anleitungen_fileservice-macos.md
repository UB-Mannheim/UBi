# FileService Zugang unter macOS einrichten

Mit dem FileService stellen wir **Beschäftigten** der Universität Mannheim einen universitätsinternen Speicherplatz zur Verfügung, der online weltweit für PCs und Laptops zugänglich ist.

**Wichtiger Hinweis:** Die Nutzung des File-Service ist im universitären Netz oder, bei Zugriff von außerhalb, mit einem VPN-Client möglich.

### Voraussetzungen

Für die Einrichtung benötigen Sie:

- Ihre [Uni-ID](https://www.uni-mannheim.de/it/anleitungen/uni-id/)
- Das dazugehörige Passwort

### Verbindung mit dem neuen Pfad (macOS)

Folgen Sie diesen Schritten, um sich mit dem Server zu verbinden:

**Schritt 1: Finder öffnen**
Wählen Sie den Finder aus und navigieren Sie über „Gehe zu“ den Punkt „Mit Server verbinden…“ aus. Alternativ können Sie das Tastenkürzel **+K** verwenden.

**Schritt 2: Serveradresse eingeben**
Geben Sie unter Serveradresse folgenden Pfad ein und klicken Sie auf „Verbinden“:
`smb://nas.uni-mannheim.de/uni-shares`

**Schritt 3: Authentifizierung**
Wählen Sie unter „Verbinden als“ die Option „Registrierter Benutzer“ aus. Geben Sie anschließend Ihre Uni-ID und das dazugehörige Passwort ein.

Klicken Sie auf „Verbinden“. Sie sind nun mit dem Server verbunden und haben Zugriff auf alle für Sie freigegebenen Shares.
