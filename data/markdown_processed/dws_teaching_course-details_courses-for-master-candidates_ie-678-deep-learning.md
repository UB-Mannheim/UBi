# IE 678: Deep Learning (FSS 2026)

This course provides an advanced study of deep learning architectures and techniques, building upon foundational knowledge of machine learning.

## 📚 Course Details

- **Type:** Lecture (in presence), exercises, assignments (6 ECTS points)
- **Prerequisites:** IE 675b Machine Learning or equivalent
- **Instructors:**
  - [Prof. Dr.-Ing. Margret Keuper](https://www.uni-mannheim.de/dws/people/professors/prof-dr-ing-margret-keuper/)
  - [Simon Forbat](https://www.uni-mannheim.de/dws/people/researchers/phd-students/simon-forbat/)
- **Schedule:** Both the lecture (Tuesday) and tutorial (Wednesday) start in the first week.
- **Materials:** Lecture notes, exercises, assignments, and supplementary material are available in [ILIAS](https://ilias.uni-mannheim.de/goto.php/crs/1719793) (requires registration via Portal²).

## 🧠 Content Overview

Machine learning involves building computer systems that improve with experience, alongside the study of learning processes and algorithms for prediction or knowledge extraction from data. This course focuses specifically on deep learning, covering both basic and advanced topics.

**Tentative Topics Include:**

- Feedforward neural networks
- Backpropagation and parameter optimization
- Machine learning systems
- Training techniques for deep learning models
- Recurrent neural networks (RNNs)
- Convolutional neural networks (CNNs)
- Attention and Transformers
- Deep learning for graphs
- Deep generative modelling
