# Mailman: Verwaltung von Mailinglisten

Mailman ist ein Programm zur Verwaltung von Mailinglisten. Es ermöglicht die Erstellung eigener Mailinglisten, um Nachrichten an mehrere Empfänger\*innen gleichzeitig zu versenden. Mailman ist daher besonders geeignet für den Massenversand von E-Mails.

## Warum ist das Einrichten von Mailinglisten sinnvoll?

Wenn E-Mails an eine große Anzahl von Empfänger\*innen versendet werden, besteht die Gefahr, dass diese Nachrichten durch unsere Mailsysteme oder auf Empfangsseite als Spam eingestuft, limitiert oder ganz abgewiesen werden.

Um dieses Problem zu vermeiden, wird das Einrichten einer Mailingliste empfohlen, sobald ein Massenversand an viele Empfänger\*innen geplant ist.
