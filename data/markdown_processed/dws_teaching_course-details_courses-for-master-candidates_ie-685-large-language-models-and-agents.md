# Large Language Models and Agents (IE-685)

This course introduces students to Large Language Models (LLMs) and LLM agents, teaching them how to employ these technologies within various applications. The course covers the potential of LLMs (such as ChatGPT, Claude, Llama, Gemini, and Mistral) to enable new applications and significantly improve existing systems.

## 📚 Kursinhalte und Lernziele

The curriculum provides comprehensive knowledge about the principles of training LLMs and their practical application.

**The course covers:**

- Introduction to LLMs and LLM Agents
- Prompt Engineering techniques
- LLM-based agents and agent frameworks
- Tool use and environment interaction
- Retrieval Augmented Generation (RAG)
- Context engineering for LLM agents
- Safety and security of LLM agents
- Evaluation of LLMs and LLM agents

**Learning Outcomes:**
Participants will gain the ability to:

- Identify opportunities for employing LLMs in business applications.
- Apply advanced prompt engineering techniques.
- Solve complex tasks using agent frameworks.
- Apply knowledge in team projects, culminating in a written report and an oral presentation.
