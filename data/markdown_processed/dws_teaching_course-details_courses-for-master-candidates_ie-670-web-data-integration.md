# Web Data Integration (HWS2025)

Data integration is a critical challenge in modern IT projects. Data scientists often spend a significant portion of their time (estimated at 80%) on data integration and preparation. In an enterprise context, data integration techniques are applied whenever data from separate sources must be combined for new applications or analytical purposes.

In the context of the Web, data integration is foundational for leveraging the growing number of publicly-accessible data sources. This enables applications such as product comparison portals, job search platforms, and advanced finance and real-estate data analysis.

## Course Overview

The course provides students with hands-on experience and theoretical knowledge in integrating and cleansing data from large sets of heterogeneous data sources.

**Key Topics Covered:**

- Heterogeneity and Distributedness
- The Data Integration Process
- Structured Data on the Web
- Data Exchange Formats
- Data Quality and Data Profiling
- Schema Mapping and Data Translation
- Identity Resolution
- Data Fusion

### Course Components and ECTS

The course is divided into two parts: a lecture and accompanying practical projects.

- **Lecture (IE670):** Covers the theory and methods of web data integration.
  - **Credits:** 3 ECTS (includes a written exam).
- **Projects (IE683):** Students gain practical experience by applying web data integration methods to a real-world use case of their choice. Work is done in teams and results are presented via a written report and an oral presentation.
  - **Credits:** 3 ECTS (70% project report, 30% presentation).

> **Recommendation:** While the lecture and project can be attended in separate years, it is highly recommended to attend both in the same semester, as the schedules are aligned.

### Requirements

- Programming skills in **Python** and experience with the **pandas** and **scikit-learn** libraries are required for the exercises and projects.

### Participation Details

- **IE670 (Lecture):** Open to all, but registration in Portal2 is required.
