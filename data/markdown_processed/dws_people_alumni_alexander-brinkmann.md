# Dr. Alexander Brinkmann

**Alumni Researcher**

## 🔬 Forschungsinteressen

Die Forschung von Dr. Brinkmann konzentriert sich auf die Schnittstelle von künstlicher Intelligenz und Datenmanagement:

- **Data Search using Deep Learning:** Entwicklung von Suchstrategien unter Einsatz von Deep Learning.
- **Product Data Integration:** Integration und Verknüpfung von Produktdaten.
- **Information Extraction using Large Language Models (LLMs):** Nutzung großer Sprachmodelle zur automatisierten Extraktion von Informationen.
