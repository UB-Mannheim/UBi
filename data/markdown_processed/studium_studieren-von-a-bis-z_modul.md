# Modul

Ein Modul ist eine Lehreinheit, die aus einer oder mehreren Lehrveranstaltungen besteht.

### Arten von Modulen

- **Pflichtmodule:** Diese Module müssen im Rahmen des Studiums belegt werden.
- **Wahlpflichtmodule:** Diese Module können aus einem thematisch eingegrenzten Bereich gewählt werden.
