## Projektbeschreibung

Im Projekt LOC-DB werden auf Basis der Linked-Data-Technologie Werkzeuge und Prozesse entwickelt, mit denen Bibliotheken an einer offenen, verteilten Infrastruktur zur Erschließung von Zitationen mitwirken können. Ziel ist es, durch weitgehende Automatisierung der Erfassungsprozesse einen Mehrwert für die wissenschaftliche Recherche zu schaffen, indem Zitationsbeziehungen routinemäßig erfasst, gespeichert und im Semantic Web verfügbar gemacht werden. Aufwand, bearbeitete Menge und erzielte Qualität werden in einer Kosten-Nutzen-Abschätzung dokumentiert.

Das Projekt nutzt bekannte Verfahren der Informationsextraktion und entwickelt diese für die automatisierte Erfassung beliebiger Literaturverzeichnisse weiter. Die erfassten Rohdaten werden mit vorhandenen Datenquellen abgeglichen und verknüpft. Es werden Lösungen erarbeitet, um die aufbereiteten Daten in bibliothekarische Präsentationssysteme einzubinden. Das System ist so konzipiert, dass es sowohl in einzelnen Bibliotheken als auch in Verbundanwendungen einsetzbar ist.

## Software & Daten

Alle entwickelte Software ist öffentlich auf GitHub verfügbar: [https://github.com/locdb](https://github.com/locdb)

**Hauptkomponenten:**

- Backend: [loc-db](https://github.com/locdb/loc-db)
- Front-end: [locdb-frend](https://github.com/locdb/locdb-frend)
- Referenzextraktor: [automatic-reference-extractor](https://github.com/locdb/automatic-reference-extractor)

**Entstandene Daten:**

- Daten sind in den [OpenCitations](https://opencitations.net/)-Korpus eingeflossen.
