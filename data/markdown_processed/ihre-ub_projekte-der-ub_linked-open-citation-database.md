# Linked Open Citation Database (LOC-DB)

Die [Linked Open Citation Database (LOC-DB)](https://www.bib.uni-mannheim.de/ihre-ub/projekte-der-ub/linked-open-citation-database/) ist ein Projekt, das von der Deutschen Forschungsgemeinschaft (DFG) gefördert wurde und von 2016 bis 2018 lief. Es zielt darauf ab, eine Linked Open Data-Datenbank zur Erschließung von Zitationsbeziehungen elektronischer und gedruckter Medien zu erstellen.

## Projektziele und -partner

Das Projekt wird von Dr. Annette Klein und Dr. Philipp Zumstein geleitet und arbeitet mit Partnern wie Prof. Dr. Andreas Dengel (DFKI Kaiserslautern), Prof. Dr. Kai Eckert (Hochschule der Medien Stuttgart) und Prof. Dr. Ansgar Scherp (ZBW) zusammen. Ziel ist es, Werkzeuge und Prozesse zu entwickeln, die Bibliotheken in die Lage versetzen, an einer offenen, verteilten Infrastruktur zur Erschließung von Zitationen mitzuwirken. 

Durch Automatisierung der Erfassungsprozesse soll ein wesentlicher Mehrwert für die wissenschaftliche Recherche geschaffen werden. Die Zitationsbeziehungen werden erfasst, gespeichert und im Semantic Web verfügbar gemacht. 

## Software und Daten

Die entwickelte Software ist auf [GitHub](https://github.com/locdb) öffentlich zugänglich. Die Hauptkomponenten des LOC-DB-Systems umfassen:

- [Backend: loc-db](https://github.com/locdb/loc-db)
- [Frontend: locdb-frend](https://github.com/locdb/locdb-frend)
- [Referenzextraktor: automatic-reference-extractor](https://github.com/locdb/automatic-reference-extractor)

Die aus dem Projekt entstandenen Daten sind in den [OpenCitations](https://opencitations.net/) Korpus eingeflossen. Weitere Daten sind:

- [Labeled Reference Data from LOC-DB Project (2018)](https://doi.org/10.7801/268)
- [Additional Labeled Reference Data from LOC-DB Project (2019)](https://doi.org/10.7801/283)
