# Data Analytics and Machine Learning Research Profile

This profile outlines the research interests, publications, and teaching activities related to data analytics, machine learning, and large-scale data processing, primarily associated with Prof. Dr. Rainer Gemulla.

## 🔬 Research Interests

The core research focus lies in advanced data processing and machine learning techniques, specifically:

- **Machine Learning with Structured Data:** Utilizing techniques for relational data.
- **Machine Learning with Semi-Structured Data:** Focusing on multi-relational graphs.
- **Knowledge Integration:** Combining structured and semi-structured data with unstructured knowledge (such as text).
- **Scalability:** Developing efficient and scalable methods and systems for data-intensive processing.

## 💻 Data and Software Resources

The group develops and utilizes various open-source tools and corpora for research:

- **Knowledge Graph & Embedding:**
  - [GraSH](https://github.com/uma-pi1/GraSH): Multi-fidelity HPO for graph learning.
  - [DistKGE](https://github.com/uma-pi1/dist-kge): A knowledge graph embedding library for multi-GPU and multi-machine training.
  - [LibKGE](https://github.com/uma-pi1/kge): A knowledge graph embedding library.
  - [AdaPM](https://github.com/alexrenz/AdaPM): A fully adaptive parameter manager.
- **Information Extraction & NLP:**
  - [OPIEC](https://www.uni-mannheim.de/dws/research/resources/opiec/): An open information extraction corpus.
  - [MinIE](https://www.uni-mannheim.de/dws/research/resources/minie/): Open information extractor (successor to [ClausIE](https://www.mpi-inf.mpg.de/departments/databases-and-information-systems/software/clausie/)).
  - [CORE](https://www.uni-mannheim.de/dws/research/resources/core/): Context-aware open relation extraction with factorization machines.
  - [FINET](http://people.mpi-inf.mpg.de/~corrogg/code/finet/index.html): Context-aware fine-grained named entity typing.
  - [Werdy](http://people.mpi-inf.mpg.de/~corrogg/code/finet/index.html): Recognition and Disambiguation of Verbs and Verb Phrases.
  - [ClausIE](https://www.mpi-inf.mpg.de/departments/databases-and-information-systems/software/clausie/): Clause-Based Open Information Extraction.
- **Optimization & Mining:**
  - [Lapse](https://github.com/alexrenz/lapse-ps/): A parameter server with dynamic parameter allocation.
  - [DSGDpp](https://github.com/uma-pi1/DSGDpp): Various parallel algorithms for matrix factorization.
  - [DESQ](https://www.uni-mannheim.de/dws/research/resources/desq/): Frequent sequence mining with subsequence constraints.
  - [Rounding rank](https://www.uni-mannheim.de/dws/research/resources/rounding-rank/): Algorithms for computing rounding-rank decompositions.
  - [LEMP](https://www.uni-mannheim.de/dws/research/resources/lemp/): Fast Retrieval of Large Entries in a Matrix Product.
  - [LASH](http://uma-pi1.github.io/lash/): Large-Scale Sequence Mining with Hierarchies.
  - [MG-FSM](http://uma-pi1.github.io/mgfsm/): Large-Scale Frequent Sequence Mining.
