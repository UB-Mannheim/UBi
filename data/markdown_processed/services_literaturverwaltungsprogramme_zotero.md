# Zotero – Literaturverwaltungssoftware an der Universitätsbibliothek Mannheim

[Zotero](https://www.bib.uni-mannheim.de/services/literaturverwaltungsprogramme/zotero/) ist eine kostenlose Open-Source-Software zur Literaturverwaltung, die unter Windows, macOS und Linux genutzt werden kann.

## Funktionen

- Bibliographische Daten während der Internetrecherche speichern
- Literaturnachweise sammeln, ordnen, mit Tags versehen und annotieren
- Volltexte speichern und durchsuchen
- Zitationen in Microsoft Word, LibreOffice und GoogleDocs einfügen; automatische Erstellung von Literaturverzeichnissen
- Auswahl und Anpassung von Zitationsstilen
- RSS-Feeds abonnieren

Optional kann ein Online-Benutzerkonto angelegt werden, um:

- Von jedem PC online auf die eigenen Daten zuzugreifen
- Daten zwischen verschiedenen Geräten zu synchronisieren
- Kollaborativ in Gruppen zu arbeiten
