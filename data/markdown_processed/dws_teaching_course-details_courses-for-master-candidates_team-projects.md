## 📚 Past Projects Archive

### Web2Table: Using LLMs to Generate Tables about Arbitrary Topics from the Web (HWS2024)

- **Description:** This project addresses the need to organize scattered online information into structured, user-friendly tables. It explores how effectively LLMs can automate the table creation process, combining information from multiple sources into a comprehensive format for analysis.
- **Supervisors:**
  - [Alexander Brinkmann](https://www.uni-mannheim.de/dws/people/alumni/alexander-brinkmann/)
  - [Christian Bizer](https://www.uni-mannheim.de/dws/people/alumni/alexander-brinkmann/)
- **Resources:**
  - [Kick-off Slides (PDF, 1020 kB)](https://www.uni-mannheim.de/media/Einrichtungen/dws/Files_Teaching/Team_Projects/Web2Table/TeamProject_HWS2024_KickOff.pdf)

### Benchmarking Function-Calling Enabled LLMs (HWS2023)

- **Description:** Focuses on building a benchmark to test the new function-calling ability of LLMs, helping them overcome issues like advanced reasoning failures and hallucination. The benchmark investigates how function descriptions and query formulations affect the performance of models like ChatGPT and GPT-4.
- **Supervisors:**
  - [Keti Korini](https://www.uni-mannheim.de/dws/people/researchers/phd-students/keti-korini/)
  - [Christian Bizer](https://www.uni-mannheim.de/dws/people/professors/prof-dr-christian-bizer/)
- **Resources:**
  - [Kick-off Slides (PDF, 2 MB)](https://www.uni-mannheim.de/media/Einrichtungen/dws/Files_Teaching/Team_Projects/TableAnnotationUsingDeepLearning/TeamProject_HWS2023_KickOff_Slides.pdf)
  - [Project Report](https://data.dws.informatik.uni-mannheim.de/Web-based-Systems-Group/StudentProjects/2024/Benchmarking%20Function-Calling%20Enabled%20LLMs/)

### Table Annotation using Deep Learning (HWS2022)

- **Description:** Investigates using Transformer-based deep learning methods for table annotation—the process of annotating a table with terms from a knowledge graph or database schema. This is a key prerequisite for data search and knowledge graph completion.
- **Supervisors:**
  - [Keti Korini](https://www.uni-mannheim.de/dws/people/researchers/phd-students/keti-korini/)
  - [Christian Bizer](https://www.uni-mannheim.de/dws/people/professors/prof-dr-christian-bizer/)

### Data Search using Deep Learning (FSS2021)

- **Description:** Explores Deep Learning techniques (like transformers) for table search and table augmentation. Students use a large table corpus extracted from the web, utilizing tools such as Elastic Search and Huggingface Transformers.
- **Supervisors:**
  - [Christian Bizer](https://www.uni-mannheim.de/dws/people/professors/prof-dr-christian-bizer/)
  - [Alexander Brinkmann](https://www.uni-mannheim.de/dws/people/alumni/alexander-brinkmann/)
- **Resources:** [Short Project Presentation (PDF, 113 kB)](https://www.uni-mannheim.de/media/Einrichtungen/dws/Files_Teaching/Team_Projects/Data_Search_using_Deep_Learning/Project-Bizer-FSS2021.pdf)

### Integrating Product Specifications from the Web (HWS2019)

- **Description:** Addresses the challenge of deriving product catalogs from the Web by matching offers from different shops and extracting, aligning, and normalizing product features. The goal is to build a comprehensive product catalogue using millions of web offers.
- **Supervisors:**
  - [Christian Bizer](https://www.uni-mannheim.de/dws/people/professors/prof-dr-christian-bizer/)

### Active and Online Learning – Your Personal Assistant for Lifestyle Improvement

- **Description:** Focuses on using inertial sensors (Accelerometer, Gyroscope) to recognize physical activities (e.g., climbing stairs, walking). The goal is to develop an Android app that learns and updates a classification model online while interacting with the user regarding uncertain results.
- **Supervisors:**

### Topic Monitoring in the Pharmaceutical Industry

- **Description:** Uses social media (Facebook, Twitter) to analyze public perception of companies and products. Key questions include: (1) what topics are discussed, (2) what topics are trending/declining, and (3) are there geographical differences in coverage.
- **Supervisors:**
  - [Heiko Paulheim](https://www.uni-mannheim.de/dws/people/professors/prof-dr-heiko-paulheim/)
  - [Sven Hertling](https://www.uni-mannheim.de/dws/people/professors/dr-sven-hertling/)

### Knowledge Extraction from WikiFarms

- **Description:** Aims to fill the "long tail" of large-scale public knowledge graphs (like DBpedia) using data from thousands of small-scale, specialized Wikis (like those at WikiFarms). The project investigates data quality and integration challenges.
- **Supervisors:**
  - [Heiko Paulheim](https://www.uni-mannheim.de/dws/people/professors/prof-dr-heiko-paulheim/)
  - [Sven Hertling](https://www.uni-mannheim.de/dws/people/professors/dr-sven-hertling/)

### Exploring the Future Warehouse

- **Description:** Focuses on developing a system that recognizes and optimizes the movement and actions of warehouse employees. It processes video and sensor data from smart devices (like data glasses) and results in a mobile application for logistics workers.
- **Supervisors:**
  - [Prof. Dr. Heiner Stuckenschmidt](https://www.uni-mannheim.de/dws/people/professors/prof-dr-heiner-stuckenschmidt/) (Responsible)

### FitFor4: Semantics-based Integration of Cyber-Physical Systems in the Industrial Internet

- **Description:** Tackles conceptual and technical problems in Industry 4.0 adaptation, such as high integration effort and non-compatibility between interfaces. The project constructs an Industry 4.0 compliant infrastructure based on semantic interface descriptions.
- **Supervisors:**
  - [Prof. Dr. Simone Paolo Ponzetto](https://www.uni-mannheim.de/dws/people/professors/prof-dr-simone-paolo-ponzetto/) (Responsible)
