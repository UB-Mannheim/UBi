# DESQ: System for Frequent Sequence Mining

DESQ is a general-purpose system designed for frequent sequence mining. It features a simple and intuitive pattern expression language, allowing users to express various pattern mining tasks, and provides efficient algorithms for executing these mining tasks.

## 💻 Source Code and Availability

DESQ is an open-source system.

- **GitHub Repository:** The primary source code is available on [GitHub](https://github.com/rgemulla/desq).
- **Experimental Version:** The source code used in the experimental study of the ICDM'16 paper is available [here](https://github.com/rgemulla/desq).
