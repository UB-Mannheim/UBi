# Sicherheitsvorfall

## 🚨 Sicherheitsvorfall melden

Bei einem vermuteten Sicherheitsvorfall können Sie diesen über folgende Wege melden:

- Nutzen Sie [unser Formular](https://www.uni-mannheim.de/informationssicherheit/sicherheitsvorfall/sicherheitsvorfall-melden/), um den Vorfall zu melden.
- Wenden Sie sich direkt an den IT Support.

## 📚 Was ist ein Sicherheitsvorfall?

Für eine detaillierte Erklärung der Definition, Beispiele und der IT-Sicherheitsziele finden Sie weitere Informationen [hier](https://www.uni-mannheim.de/informationssicherheit/sicherheitsvorfall/was-ist-ein-sicherheitsvorfall/).

## ⚠️ Aktuelle Warnhinweise

Hier finden Sie eine Übersicht der aktuellen Warnungen vor Spam- und Phishing-Mails.

- [Hier](https://www.uni-mannheim.de/informationssicherheit/aktuelle-warnungen/)
- *Hinweis:* Diese Informationen sind nur Mitgliedern der Universität zugänglich.

## 🧑‍💻 Sprechstunde für Informationssicherheit

Für alle Beschäftigten der Universität Mannheim gibt es die Möglichkeit, an einer Sprechstunde für Informationssicherheit teilzunehmen.

- [Anmeldung via Formular](https://www.uni-mannheim.de/informationssicherheit/sicherheitsvorfall/sprechstunde-fuer-informationssicherheit/)
