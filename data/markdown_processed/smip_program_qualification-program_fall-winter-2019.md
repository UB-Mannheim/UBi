# SMiP PhD Qualification Program: Fall/Winter 2019 Schedule

This document outlines the mandatory and elective components of the SMiP PhD Qualification Program for the Fall/Winter 2019 cohort.

  - **Time:** 3 p.m. – 6 p.m. (Official part), followed by an informal get-together.
  - **Location:** Mannheim (B6, 30–32, Room 309)
