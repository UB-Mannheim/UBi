# Online Random Forest Classifier

This document describes an implementation of an online random forest classifier written in Java.

The architecture and implementation details are based on the work of Amir Saffari, but the provided version includes several enhancements.

## Features and Support

The classifier supports advanced machine learning metrics and operational requirements:

- **Quality Measurements:** Information Gain and Gini Index.
- **Feature Threshold:** Supports random and incremental feature thresholding.
- **Concurrency:** Full threading support.
- **Language Compatibility:** Java 1.7 and newer (compatible with Android).
- **Data Handling:** Export/Import functionality (coming soon).
