# Microsoft Copilot Chat: Überblick und Nutzung

Microsoft Copilot Chat ist ein sicherer KI-Chat, der verschiedene Funktionen bietet, darunter:

- Zugriff auf das Internet
- Zusammenfassen und Generieren von Texten
- Generieren von Bildern
- Analyse einfacher Daten

## Copilot Chat vs. M365 Copilot

Es ist wichtig zu beachten, dass Copilot Chat nicht identisch mit M365 Copilot ist. Die beiden unterscheiden sich sowohl im Funktionsumfang als auch in der Anbindung.

Für detaillierte Informationen zu den Unterschieden verweisen wir auf die offiziellen [Seiten von Microsoft](https://learn.microsoft.com/en-us/copilot/overview#differences-between-copilot-chat-and-microsoft-365-).

## So nutzen Sie Copilot Chat

Sie können Copilot Chat über verschiedene Kanäle zugänglich machen:

- **Webbrowser:** Über die dedizierte Chat-Seite: [https://m365.cloud.microsoft/chat/](https://m365.cloud.microsoft/chat/)
- **Microsoft 365 Copilot Desktop App:** Die Anwendung kann über die [Microsoft Apps](https://m365.cloud.microsoft/apps) installiert werden. Nach dem Öffnen der Anwendung ist der Copilot Chat direkt auf der Startseite verfügbar.
- **In Office-Anwendungen (z.B. Word):** Das Icon von Copilot Chat finden Sie innerhalb der jeweiligen Anwendung oben rechts.

Um den Chat zu starten, geben Sie Ihren Prompt ein.
