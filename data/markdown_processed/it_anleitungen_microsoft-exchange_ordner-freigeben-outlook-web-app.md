# Ordner freigeben und einbinden in Outlook Web App (OWA)

Bevor Sie fortfahren, melden Sie sich bitte im Webmailer OWA (Outlook Web App) unter [https://exchange.uni-mannheim.de](https://exchange.uni-mannheim.de) mit Ihrer Uni-ID und dem dazugehörigen Passwort an.

## 📂 Einen Ordner freigeben

Um einen Ordner für andere Benutzer zugänglich zu machen, gehen Sie bitte wie folgt vor:

1. Führen Sie einen Rechtsklick auf den gewünschten Ordner durch.
1. Wählen Sie im Kontextmenü „Berechtigungen...“.
1. Im Fenster „Berechtigungen für den Ordner [Name]“ klicken Sie auf das „+“-Zeichen.
1. Im Fenster „Berechtigungen hinzufügen“ geben Sie den Namen oder die E-Mail-Adresse der Person ein, die Zugriff erhalten soll, und bestätigen Sie mit einem Linksklick auf „Hinzufügen“.
1. Das Fenster „Berechtigungen für den Ordner [Name]“ wird erneut angezeigt. Markieren Sie den Namen der berechtigten Person und wählen Sie die gewünschten Berechtigungen aus oder wählen Sie eine Berechtigungsstufe aus.
   - *Hinweis:* Eine Übersicht über die Berechtigungsstufen finden Sie auf der offiziellen Microsoft Website unter [Microsoft Support](https://support.microsoft.com/de-de/office/options-for-sharing-and-delegating-folders-in-outlook-for-mac-480d8054-68ce-4150-ba1e-b9b7f2fc4ce5).
1. Klicken Sie auf „OK“, um die Berechtigungen zu speichern.

## 📥 Freigegebenen Ordner/Postfach einbinden

Um einen Ordner, der Ihnen von einem anderen Benutzer freigegeben wurde, in Ihrem Postfach anzuzeigen, gehen Sie bitte wie folgt vor:

1. Führen Sie einen Rechtsklick auf Ihr gesamtes Postfach (Ihren Namen) durch.
1. Wählen Sie im Kontextmenü „Freigegebenen Ordner hinzufügen...“.
1. Im erscheinenden Fenster „Freigegebenen Ordner hinzufügen“ geben Sie den Namen oder die E-Mail-Adresse der Person ein, die den Ordner für Sie freigegeben hat, und klicken Sie auf „Hinzufügen“.
1. Der freigegebene Ordner wird anschließend unter dem Namen der Person, die ihn freigegeben hat, in Ihrem Outlook angezeigt.
