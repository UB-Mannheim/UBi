# Installation von Office 365 unter macOS

**Wichtiger Hinweis zur Kompatibilität:**
Bitte beachten Sie, dass Office 365 nur unter der aktuellsten Version von macOS oder einer der beiden Vorgängerversionen unterstützt wird.

**Besondere Vorgehensweise bei Office Professional Plus 2016/2019:**
Falls auf Ihrem Apple Mac bereits eine Version von Office Professional Plus 2016/2019 installiert ist und die vorherige Lizenz mit dem „Volume Serializer“ aktiviert wurde, müssen Sie bei der Installation von Office 365 unbedingt die folgenden Schritte befolgen:

### Installationsschritte

**Schritt 1: Anmeldung im Office Portal**
Melden Sie sich mit Ihrer Uni-ID unter [https://portal.office.com](https://portal.office.com) an:

- **E-Mail:** Uni-ID@ad.uni-mannheim.de (Beispiel: mmuster@ad.uni-mannheim.de)
- **Kennwort/Passwort:** Das Ihrer Uni-ID.
- Bestätigen Sie die Rückfrage „Angemeldet bleiben?“ mit **Ja**.

**Schritt 2: Download der Setup-Datei**
Laden Sie rechts oben die Setup-Datei für Office 365 herunter.

**Schritt 3: Ausführen der Setup-Datei**
Speichern Sie die Datei auf Ihrem Mac ab und führen Sie sie anschließend aus.

**Schritt 4: Installation**
Installieren Sie Office 365, indem Sie den Anweisungen des Setup-Programms folgen.

**Schritt 5: Entfernen der alten Lizenz**
Laden Sie das [Microsoft Office License Removal Tool](https://support.microsoft.com/de-de/office/entfernen-von-office-lizenzdateien-auf-einem-mac-b032c0f6-a431-4dad-83a9-6b727c03b193) für Mac herunter.

**Schritt 6: Ausführen des Removal Tools**
Installieren Sie das Tool und führen Sie es aus. Bestätigen Sie, dass Sie die vorhandene Lizenz für Microsoft Office entfernen möchten.

**Schritt 7: Erste Anmeldung und Aktivierung**
Starten Sie erst jetzt eine der neu installierten Office 365 Anwendungen (z. B. Microsoft Word). Sie werden direkt zur Anmeldung aufgefordert. Authentifizieren Sie sich wie in Schritt 1 beschrieben mit:

- **E-Mail:** Uni-ID@ad.uni-mannheim.de
- **Kennwort:** Das zugehörige Kennwort.

*Ergebnis:* Ihre Installation von Office 365 wird nun bei Microsoft auf diesem Endgerät registriert und die zugehörige Lizenz installiert. Sie können Office 365 verwenden.
