# Universitätsarchiv Mannheim: Katalog und Bestände

Das Universitätsarchiv Mannheim bietet eine umfangreiche Sammlung historischer Quellen und Publikationen zu verschiedenen Themenbereichen. Die Bestände sind nach verschiedenen Signaturen und Themengebieten geordnet.

## 📚 Themenübersichten der Bestände

Für eine thematische Orientierung sind folgende PDF-Übersichten verfügbar:

- **Universitätsgeschichte:** Eine Übersicht über alle Bände zum Thema Universitätsgeschichte finden Sie [hier (PDF, 81 kB)](https://www.uni-mannheim.de/media/Einrichtungen/universitaetsarchiv/Handbibliothek_Universitaetsgeschichte.pdf).
- **Druckschriften der Universität:** Eine Übersicht über die Druckschriften finden Sie [hier (PDF, 86 kB)](https://www.uni-mannheim.de/media/Einrichtungen/universitaetsarchiv/Handbibliothek_Druckschriften_der_Universitaet.pdf).
- **Hochschulpolitik:** Eine Übersicht über alle Bände zur Hochschulpolitik finden Sie [hier (PDF, 186 kB)](https://www.uni-mannheim.de/media/Einrichtungen/universitaetsarchiv/Handbibliothek_Hochschulpolitik.pdf).
- **Archive:** Eine Übersicht über alle Bände zum Thema Archive finden Sie [hier (PDF, 40 kB)](https://www.uni-mannheim.de/media/Einrichtungen/universitaetsarchiv/Handbibliothek_Archive.pdf).
- **Mannheim:** Eine Übersicht über alle Bände zum Thema Stadt Mannheim finden Sie [hier (PDF, 67 kB)](https://www.uni-mannheim.de/media/Einrichtungen/universitaetsarchiv/Handbibliothek_Mannheim.pdf).
- **Verschiedenes:** Eine Übersicht über alle Bände zum Thema Verschiedenes finden Sie [hier (PDF, 50 kB)](https://www.uni-mannheim.de/media/Einrichtungen/universitaetsarchiv/Handbibliothek_Verschiedenes.pdf).
