# Xenia Paul

Xenia Paul ist Stipendiatin des Elite-Sports-Stipendiums der Region Rhein-Neckar.

### Profilübersicht

| Kriterium | Details |
| :--- | :--- |
| **Förderungsbeginn** | HWS 2022/2023 |
| **Sportart** | Ringen |
| **Studium** | Betriebswirtschaftslehre (BWL) |
| **Verein** | SRC 1896 Viernheim |

### Sportliche Erfolge

Die größten sportlichen Erfolge von Xenia Paul umfassen:

- **3. Platz** Deutsche Meisterschaft Juniorinnen (2021)
- **1. Platz** Deutsche Meisterschaft Kadetinnen (2019)
- **3. Platz** Deutsche Meisterschaft Kadetinnen (2018)
