# Support-Anfrage: Beschreibung Ihres Anliegens

Bitte skizzieren Sie Ihr Anliegen detailliert, indem Sie folgende Punkte berücksichtigen. Die Anforderungen variieren je nachdem, ob es sich um eine Datenanfrage oder eine Befragung handelt.

## 📝 Bei einer Befragung (Umfrage)

Wenn Sie Informationen durch eine Befragung gewinnen möchten, geben Sie bitte Auskunft über:

- **Zielsetzung:** Welche Informationen möchten Sie mit der Befragung gewinnen und wofür benötigen Sie die Ergebnisse?
- **Mandat:** Liegt für Ihr Vorhaben bereits ein konkreter Arbeitsauftrag vor (z. B. seitens des Rektorats oder der Dezernatsleitung)?
- **Zielgruppe:** Welche Personengruppe/n soll/en befragt werden (z. B. Studierende, Lehrende)?
- **Zeitrahmen:** Bis wann benötigen Sie die Ergebnisse?
- **Material:** Gibt es bereits einen ersten Entwurf des Fragebogens?

## 📊 Bei einer Datenanfrage

Wenn Sie spezifische Daten benötigen, geben Sie bitte Auskunft über:

- **Datenbedarf:** Welche Daten werden benötigt?
- **Zweck:** Wofür werden die Daten benötigt? Bitte skizzieren Sie kurz den Anlass bzw. den Zweck Ihrer Datenanfrage.
- **Zeitrahmen:** Bis wann benötigen Sie die Daten?
