# E-Mail-Weiterleitung in Ihrem Uni-Postfach einrichten

Sie können eingehende Nachrichten in Ihrem Exchange-Postfach automatisch auf eine andere universitäre E-Mail-Adresse weiterleiten.

## ⚠️ Wichtige Hinweise und Einschränkungen

Bitte beachten Sie die folgenden Regeln, bevor Sie mit der Einrichtung beginnen:

**Für Beschäftigte der Universität Mannheim:**
Es ist untersagt, Weiterleitungen an private Mailkonten oder universitätsexterne Server einzurichten.

**Für Studierende:**
Sie sind verpflichtet, Ihr universitäres E-Mail-Postfach regelmäßig abzurufen, da Benachrichtigungen der Universität Mannheim ausschließlich an dieses gesendet werden. Wir raten dringend davon ab, Weiterleitungen an private Mailaccounts zu nutzen.

### Nachteile von Weiterleitungen an private Adressen

1. **SPAM-Risiko:** Bei der Weiterleitung großer Mengen von E-Mails an einen externen Anbieter (z. B. GMX) kann dies als SPAM-Angriff gewertet werden. Dies kann dazu führen, dass die Universität Mannheim als Absender gesperrt wird und die E-Mails nicht zugestellt werden.
1. **Fehlzustellung:** E-Mails können durch Tippfehler in der Zieladresse nicht zugestellt werden, wodurch wichtige Benachrichtigungen verpasst werden.

## ⚙️ Schritt-für-Schritt-Anleitung zur Einrichtung

Um eine Weiterleitung einzurichten, gehen Sie bitte wie folgt vor:

1. **Zugriff:** Rufen Sie die [Outlook Web App (OWA)](https://exchange.uni-mannheim.de) auf.
1. **Anmelden:** Geben Sie Ihre Uni-ID unter „Benutzername“ und Ihr zugehöriges Passwort unter „Kennwort“ ein und klicken Sie auf „Anmelden“.
1. **Einstellungen öffnen:** Klicken Sie in Ihrem Postfach rechts oben auf das Zahnrad-Symbol, um die Einstellungen zu öffnen, und wählen Sie anschließend „Optionen“.
1. **Regeln erstellen:** Klicken Sie im Menü auf „Posteingangs- und Aufräumregeln“ und anschließend auf das „+“-Symbol.
1. **Regel-Auswahl:** Wählen Sie unter „Wenn die Nachricht eintrifft […]“ die Option **„Auf alle Nachrichten anwenden“** aus.
1. **Aktion definieren:** Wählen Sie unter „Alle folgenden Aktionen ausführen“ die Option „Weiterleiten, umleiten oder senden“ aus.
   - Wählen Sie **„Nachricht weiterleiten an…“**.
     - *Achtung:* Dies leitet die Nachricht an das Zielpostfach weiter, wobei der ursprüngliche Absender durch das weiterleitende Postfach ersetzt wird.
     - **Wählen Sie NICHT** die Option „Nachricht umleiten“.
1. **Zieladresse eintragen:** Sie können die gewünschte Zieladresse auf zwei Arten eintragen:
   - Geben Sie einen Namen ein und klicken Sie auf einen der Kontaktvorschläge oder durchsuchen Sie das Exchange-Adressbuch.
   - ODER, falls die Adresse nicht im Adressbuch zu finden ist (häufig bei Studierenden/Externen): Geben Sie die vollständige Adresse direkt ein.
   - *(Hinweis: Bei der Eingabe der Zieladresse kann die Option zum automatischen Löschen der Nachricht nach dem Weiterleiten aktiviert werden.)*
1. **Aktion hinzufügen:** Klicken Sie auf den Button „Aktion hinzufügen“. Wählen Sie unter „Verschieben, kopieren oder löschen“ die Option **„Nachricht löschen“** aus.
1. **Abschluss:** Geben Sie abschließend einen Namen für die Weiterleitungsregel ein (falls der automatisch generierte Name nicht ausreichend ist) und klicken Sie auf „OK“.

Die Regel ist nun aktiv.

## 💡 Alternativen zur Weiterleitung

Wir empfehlen, die folgenden Methoden zu nutzen, um wichtige Benachrichtigungen sicherzustellen:

- **Direkter Browser-Zugriff:** Rufen Sie Ihr Postfach direkt im Browser über die [Outlook Web App (OWA)](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/login-outlook-web-app/) auf.
- **E-Mail-Client:** Verbinden Sie Ihr Exchange-Postfach mit einem E-Mail-Client (z. B. Outlook) auf Ihrem Endgerät. [Hier finden Sie die entsprechenden Anleitungen](https://www.uni-mannheim.de/it/anleitungen/microsoft-exchange/#c259304).
