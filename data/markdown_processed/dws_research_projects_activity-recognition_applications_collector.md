# Sensor Data Collector Application

The Sensor Data Collector is an application designed for research purposes, enabling the comprehensive recording and analysis of data from various wearable sensors.

## Functionality and Capabilities

This application allows users to record data from all common built-in sensors of wearable devices. Key features include:

- **Multi-Sensor Recording:** Simultaneous recording of multiple sensors is possible.
- **Customization:** Users can specify the desired sampling rate for the recorded data.
- **Data Labeling:** The recorded data can be accurately labeled by specifying the current activity, posture, and location.
- **Visualization and Export:** Recorded data can be visualized, including live plotting, and exported in several formats (e.g., SQLite, CSV).

## Smart-Watch Integration

The system supports the integration of smart-watches, significantly enhancing data collection capabilities:

- **Dual Recording:** Combining a smartphone with at least one smart-watch allows for simultaneous sensor recording from different on-body positions.
- **Enhanced Control:** The smart-watch provides an additional interface, allowing the user to control the application (e.g., updating the current location or activity) without needing to hold the smartphone, which ensures highly accurate data labeling.
