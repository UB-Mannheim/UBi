# Introduction to Cognitive Modeling

**Instructors:**

- Chris Donkin (LMU Munich)
- Nathan Evans (University of Queensland)

**Overview:**
This course provides participants with a comprehensive overview of cognitive modeling. Using a combination of lectures, discussions, and hands-on exercises, the workshop aims to demonstrate how cognitive modeling can advance psychological theories and outlines the necessary tools to achieve this goal.

**Key Topics Covered:**

- Conceptual issues surrounding cognitive modeling (e.g., how it differs from statistical modeling and verbal theories).
- The process of moving from a verbal theory to a formal model.
- Technical methods for computational or mathematical cognitive modeling, including:
  - Parameter estimation
  - Model evaluation
  - Model selection (from both frequentist and Bayesian perspectives)

The content is consistently related back to the larger goal of using these tools to advance theory development and testing, utilizing specific cognitive models (e.g., evidence-accumulation models).

**Prerequisites:**
Participants should have:

- Experience in the use of R.
- Familiarity with basic concepts of Bayesian analysis (e.g., likelihood, prior distribution, posterior distribution).
