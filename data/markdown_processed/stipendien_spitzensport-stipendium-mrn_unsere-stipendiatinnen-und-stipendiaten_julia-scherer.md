# Julia Scherer

Julia Scherer ist Stipendiatin des Elite Sports Scholarship Rhine-Neckar Metropolitan Region.

| Detail | Information |
| :--- | :--- |
| **Förderungsbeginn** | HWS 2021/2022 |
| **Sportart** | Standardtanz |
| **Studium** | Medien- und Kommunikationswissenschaft |
| **Verein** | TSC Landau |

## Sportliche Erfolge

Julia Scherer hat in ihrem sportlichen Leben zahlreiche Erfolge erzielt, darunter:

- 3. Platz Rangliste HGR S STD (2021)
- 19. Platz Weltmeisterschaft U21 STD (2021)
- 12. Platz Deutsche Meisterschaft HGR S STD (2021)
- 2. Platz Deutsche Meisterschaft U21 (2019)
- 1. Platz Süddeutsche Meisterschaft Jugend (2019)
- 8. Platz Deutsche Meisterschaft Jugend (2019)
